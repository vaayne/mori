# Mori 键盘快捷键

Mori 使用 tmux 作为会话后端。键盘快捷键通过原生 macOS 界面管理 tmux 会话、窗口（标签页）和窗格。

## 应用

| 快捷键 | 操作 |
|----------|--------|
| `⌘⇧O` | 打开项目 |
| `⌘,` | 设置（打开 ghostty 配置） |
| `⌘⇧,` | 重新加载设置 |
| `⌘H` | 隐藏 Mori |
| `⌘⌥H` | 隐藏其他窗口 |
| `⌘Q` | 退出 Mori |

## 编辑

| 快捷键 | 操作 |
|----------|--------|
| `⌘Z` | 撤销 |
| `⌘⇧Z` | 重做 |
| `⌘X` | 剪切 |
| `⌘C` | 复制 |
| `⌘V` | 粘贴 |
| `⌘A` | 全选 |

## 标签页（tmux 窗口）

| 快捷键 | 操作 |
|----------|--------|
| `⌘T` | 新建标签页 |
| `⌘W` | 关闭窗格（最后一个窗格将关闭标签页） |
| `⌘⇧]` | 下一个标签页 |
| `⌘⇧[` | 上一个标签页 |
| `⌘1`–`⌘9` | 跳转到标签页 1–9（`⌘9` = 最后一个） |

## 窗格（tmux 窗格）

| 快捷键 | 操作 |
|----------|--------|
| `⌘D` | 右分屏 |
| `⌘⇧D` | 下分屏 |
| `⌘]` | 下一个窗格（循环） |
| `⌘[` | 上一个窗格（循环） |
| `⌘⌥↑` | 跳转到上方窗格 |
| `⌘⌥↓` | 跳转到下方窗格 |
| `⌘⌥←` | 跳转到左方窗格 |
| `⌘⌥→` | 跳转到右方窗格 |
| `⌘⌃↑` | 向上调整窗格大小 |
| `⌘⌃↓` | 向下调整窗格大小 |
| `⌘⌃←` | 向左调整窗格大小 |
| `⌘⌃→` | 向右调整窗格大小 |
| `⌘⇧↩` | 切换窗格缩放 |
| `⌘⌃=` | 均分窗格 |

## 自定义快捷键

以上列出的所有 Mori 应用快捷键（锁定的系统快捷键除外）均可通过**设置 > 键盘**完全自定义。
您可以将任何可配置的快捷键重新映射到其他组合键、取消分配，或将其重置为默认值。

锁定的快捷键——例如编辑命令（⌘Z、⌘C 等）、退出（⌘Q）、隐藏（⌘H）、最小化（⌘M）
和切换全屏（⌘⌃F）——无法更改，因为它们通过 macOS AppKit 响应链路由。

Mori 包含冲突检测功能：如果您分配的快捷键已被锁定的系统绑定使用，该分配将被阻止。
如果冲突的是另一个可配置的绑定，Mori 会发出警告并提供重新分配的选项，替换之前的绑定。
要将所有快捷键恢复为原始默认值，请使用设置 > 键盘中的**全部重置**按钮。

快捷键覆盖保存在 `~/Library/Application Support/Mori/keybindings.json` 中，
采用稀疏文件格式（仅存储修改过的绑定）。更改立即生效，无需重启应用。

> **注意：** Ghostty 终端快捷键需单独配置——请参阅下方章节。

## Ghostty 终端

与 Mori 快捷键不冲突的 Ghostty 快捷键会直接传递到终端。常见示例：

| 快捷键 | 操作 |
|----------|--------|
| `⌘K` | 清屏 |
| `⌘+` | 增大字体 |
| `⌘-` | 减小字体 |

在 `~/.config/ghostty/config` 中自定义终端快捷键。
