# Mori 键盘快捷键

Mori 使用 tmux 作为会话后端。键盘快捷键通过原生 macOS 界面管理 tmux 会话、窗口（标签页）和窗格。

## 应用

| 快捷键 | 操作 |
|----------|--------|
| `⌘⇧O` | 打开项目 |
| `⌘,` | 设置（打开 ghostty 配置） |
| `⌘⇧,` | 重新加载设置 |
| `⌘H` | 隐藏 Mori |
| `⌘⌥H` | 隐藏其他窗口 |
| `⌘Q` | 退出 Mori |

## 编辑

| 快捷键 | 操作 |
|----------|--------|
| `⌘Z` | 撤销 |
| `⌘⇧Z` | 重做 |
| `⌘X` | 剪切 |
| `⌘C` | 复制 |
| `⌘V` | 粘贴 |
| `⌘A` | 全选 |

## 标签页（tmux 窗口）

| 快捷键 | 操作 |
|----------|--------|
| `⌘T` | 新建标签页 |
| `⌘W` | 关闭窗格（最后一个窗格将关闭标签页） |
| `⌘⇧]` | 下一个标签页 |
| `⌘⇧[` | 上一个标签页 |
| `⌘1`–`⌘9` | 跳转到标签页 1–9（`⌘9` = 最后一个） |

## 窗格（tmux 窗格）

| 快捷键 | 操作 |
|----------|--------|
| `⌘D` | 右分屏 |
| `⌘⇧D` | 下分屏 |
| `⌘]` | 下一个窗格（循环） |
| `⌘[` | 上一个窗格（循环） |
| `⌘⌥↑` | 跳转到上方窗格 |
| `⌘⌥↓` | 跳转到下方窗格 |
| `⌘⌥←` | 跳转到左方窗格 |
| `⌘⌥→` | 跳转到右方窗格 |
| `⌘⌃↑` | 向上调整窗格大小 |
| `⌘⌃↓` | 向下调整窗格大小 |
| `⌘⌃←` | 向左调整窗格大小 |
| `⌘⌃→` | 向右调整窗格大小 |
| `⌘⇧↩` | 切换窗格缩放 |
| `⌘⌃=` | 均分窗格 |

## 工具

| 快捷键 | 操作 |
|----------|--------|
| `⌘G` | 打开 Lazygit |
| `⌘E` | 打开 Yazi |

## 窗口

| 快捷键 | 操作 |
|----------|--------|
| `⌘B` | 切换侧边栏 |
| `⌘⌃F` | 切换全屏 |
| `⌘M` | 最小化 |
| `⌘⇧W` | 关闭窗口 |

## 工作树

| 快捷键 | 操作 |
|----------|--------|
| `⌃Tab` | 下一个工作树 |
| `⌃⇧Tab` | 上一个工作树 |

## 命令面板

| 快捷键 | 操作 |
|----------|--------|
| `⌘⇧P` | 切换命令面板 |

## Ghostty 终端

与 Mori 快捷键不冲突的 Ghostty 快捷键会直接传递到终端。常见示例：

| 快捷键 | 操作 |
|----------|--------|
| `⌘K` | 清屏 |
| `⌘+` | 增大字体 |
| `⌘-` | 减小字体 |

在 `~/.config/ghostty/config` 中自定义终端快捷键。
